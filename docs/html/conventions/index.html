規約カタログ (Conventions)

Harmony プロジェクトで採用する規約群。@conv.* / @env.* / @secret.* 等の式言語規約、バリデーション規約、プロダクトスコープ定義を含む。

実体の JSON 正本は active workspace の <dataDir>/conventions/catalog.json に格納。本ページは人間向け参考資料。
