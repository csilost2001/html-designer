このサイトについて

本サイトは docs/spec/*.md / docs/user-guide/*.md / docs/conventions/*.md / docs/setup/*.md を Astro + Tailwind + pagefind で静的 HTML 化したものです。

Source of truth は Markdown — 本サイトの内容は docs/* の md 更新後に cd docs-site && npm run build で再生成されます (docs/html/ は build artifact、手編集禁止)。
